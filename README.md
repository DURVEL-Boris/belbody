# BèlBody — Site vitrine

Site vitrine single-page pour **BèlBody Coaching**, coaching fitness en ligne basé en Martinique.

---

## Aperçu

Landing page complète présentant l'offre de coaching, les formules tarifaires, les témoignages clients et un formulaire de réservation d'appel découverte.

## Contenu du site

- **Hero** — accroche principale avec statistiques clés
- **Le problème** — les 4 obstacles courants sans coaching
- **Ma méthode** — les 3 piliers (Santé, Esthétique, Efficacité)
- **Formules** — Essai gratuit / Starter 35€ / Pro 75€
- **Avis clients** — témoignages
- **Comment ça marche** — timeline en 4 étapes
- **FAQ** — questions fréquentes
- **Réservation** — formulaire d'appel découverte (Formspree)
- **Contact** — téléphone et email directs

| Fichier | Description |

## Stack technique

- HTML5 / CSS3 / JavaScript vanille — aucune dépendance externe
- **Google Fonts** : Fraunces (titres) + Inter (corps de texte)
- **Formspree** : backend du formulaire de réservation
- CSS custom properties, IntersectionObserver, dropdowns JS sur mesure

## Déploiement (GitHub Pages)

1. Aller dans **Settings → Pages**
2. Source : `Deploy from a branch`
3. Branch : `main` / `root`
4. Le site sera accessible à `https://<ton-username>.github.io/<nom-du-repo>/`
